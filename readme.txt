== Description ==
Integration of Kunaki and WooCommerce is a WooCommerce Extension that allows Musicians, Filmmakers, Software companies, or anyone to sell their own CDs or DVDs, through kunaki.com one at a time, on-demand, and through their own website, Using the api/xml that Kunaki provides.

== Installation ==

1. Upload the `Kunaki-woo.zip` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can configure the Plugin Settings in `Woocommerce` -> `Kunaki lite`
